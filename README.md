# AIM

To write a Python program using OpenCV to read and display an image, adjust brightness, modify contrast, and perform bitwise operations.

# SOFTWARE REQUIRED

Python 3.7 (Anaconda)

# EXPLANATION

Image processing involves manipulating pixel values to enhance or transform images. Brightness adjustment changes the intensity of pixels, while contrast modification improves the difference between light and dark regions. Bitwise operations are used to combine or manipulate images at the pixel level. OpenCV provides efficient functions to perform these operations.

# ALGORITHM

Step 1: Load an image from the local directory and display it

Step 2: Create a matrix of ones with data type float64 to adjust brightness

Step 3: Generate brighter and darker images by adding and subtracting the matrix from the original image and display them

Step 4: Modify the image contrast using scaling factors of 1.1 and 1.2 and display the results

Step 5: Split the image (boy.jpg) into Blue, Green, and Red channels and display them
